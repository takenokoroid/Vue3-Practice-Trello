# Vue3-Practice-Trello

## ここはなに？

作成時間は３日（12~15 時間）くらい

詳しくはコードみてね。issue 駆動で開発してたので履歴は[こちら](https://github.com/takenokoroid/Vue3-Practice-Trello/issues?q=is%3Aissue+is%3Aclosed)。
